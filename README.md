#Clash Royal Deck Builder

A web application that allows users to build analyze and optimize Clash Royale decks with deck analysis and suggestions.

**Created By:** Jakob Kominsky & Evan Neuman
**Course:** Web Development 330 Final Project

---

##Project Overview

This deck builder help Clash Royal players create competitive decks by providing:
- Interactive drag-and -drop card selection
- Real-time deck statistics and analysis
- Intelligent deck rating system
- Card filtering and search functionality
- Evolution card support

---

##Features

- **Interactive Deck Building**: Drag-and-drop or click to add cards to your 8-card deck
- **Card Library**: Browse all Clash Royale cards with filtering options (Ground/Air)
- **Evolution Support**: Special slots for evolution cards
- **Visual Feedback**: Cards dim when already in deck, tooltips show card details
- **Deck Statistics**: 
  - Average elixir cost calculation
  - Card type breakdown
  - Deck completeness validation
